---
title: Overview
description: Reusable React blocks for SendLit product flows.
---

`@sendlit/email-blocks` exports headless, ready-to-use React components for composing SendLit flows.

The package contains form blocks, selectors, previews, and list controls. Components use controlled props so host apps own persistence and API calls.

## Components

- `ContactFilterBuilder`
- `TriggerPicker`
- `TagEditor`
- `SequenceAnalytics`
- `SubscriberList`
- `SequenceEmailList`
- `SequenceMetaForm`
- `EmailPreview`
- `TemplateChooser`
- `SequenceEmailForm`
