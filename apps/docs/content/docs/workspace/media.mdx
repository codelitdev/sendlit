---
title: Media
description: Upload and reuse images in email content.
---

The Media library stores images that can be reused by the email editor and templates. Upload an image from **Media**, then select it from the editor’s image controls when composing a broadcast, sequence email, or template.

Media is scoped to the current team. Keep uploads optimized for email and use HTTPS-hosted assets in production. The dashboard applies a configurable maximum upload size; the default is 10 MB.
