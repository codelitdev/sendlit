---
title: First Broadcast
description: Compose, target, and send a one-off email.
---

To send a broadcast:

1. Create or import contacts.
2. Create a template or start from a system template.
3. Create a broadcast.
4. Pick the audience filter.
5. Compose the single email.
6. Send now or schedule for a future time.

Scheduled broadcasts can be cancelled until the scheduler locks the broadcast for delivery.
